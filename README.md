# ECommerce_Project_DDD 
## Description
This is a simple e-commerce project, just backend project and is an implementation on Domain Driven Design. 
### Technologies
* .Net Core 7 Web API
* EntityFramework Core
### Architecture
- Onion Architecture
* Domain Driven Design
### Patterns
* Repository Pattern
* Specification Pattern
* Unit Of Work
* Mediator Pattern
* CQRS 
* Fluent Validation
* AutoMapper
